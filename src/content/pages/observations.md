# The Architect’s Eye
Photography is my primary tool for practicing **Pattern Recognition**. In the terminal, I'm looking for the most efficient path for data; through the viewfinder, I'm looking for the most resonant path for light. Both require the same fundamental skill: the ability to see the underlying architecture before the first "brick" is laid.
