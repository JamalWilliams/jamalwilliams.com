### Astro 5.0
The engine. Zero JavaScript by default, enabling high-resolution photography to load almost instantly without plugin bloat.

### Cloudflare Pages
Global edge delivery. Commercial-friendly hosting that offers unlimited bandwidth and automated Git deployments.

### Obsidian / Markdown
The CMS. My knowledge vault is my database. I write in local Markdown, and Git handles the sync to the live site.

### Tailwind CSS
The styling. Utility-first design that allows for rapid, expressive visual systems without the weight of traditional CSS frameworks.

## Why this stack?
WordPress is powerful but heavy. Social media platforms are convenient but you don't own the data. 

By shifting to a **Static + Git + Markdown** workflow, I've stripped away the vulnerabilities and database maintenance. My sites are now safer, faster, and truly sovereign.
